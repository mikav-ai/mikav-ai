# Contributing to Mikav

Thanks for wanting to help build Mikav! This project is open to everyone — developers, designers, writers, researchers, culture enthusiasts.

## How to Contribute

### 1. Fork & Clone

```bash
git clone https://github.com/<your-username>/mikav-ai.git
cd mikav-ai
npm install
cp .env.example .env.local
npm run dev
```

### 2. Create a Branch

```bash
git checkout -b feat/your-feature-name
```

### 3. Make Your Changes

- Follow the existing code style
- Use TypeScript — no `any` unless unavoidable
- Use Tailwind CSS for styling
- Use shadcn/ui components where possible
- Keep commits small and focused

### 4. Test

```bash
npm run build
npm run lint
```

### 5. Submit a Pull Request

- Open a PR against `main`
- Describe what you changed and why
- Link any related issues

## What Can You Contribute?

- **Code** — Features, bug fixes, performance improvements
- **Data** — Kerala culture datasets, art form documentation
- **Design** — UI improvements, illustrations, icons
- **Docs** — Improve guides, fix typos, add examples
- **Ideas** — Open an issue to suggest features or improvements

## Guidelines

- Be respectful — follow our [Code of Conduct](CODE_OF_CONDUCT.md)
- Keep PRs focused — one feature or fix per PR
- Write clear commit messages
- Don't push directly to `main`

## Need Help?

Open an issue or reach out at **hello@mikav.in**.

We're glad you're here.
